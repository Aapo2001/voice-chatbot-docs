# Voice Chatbot — Documentation

Documentation website for the [voice-chatbot](https://pypi.org/project/voice-chatbot/) project.

Built with React + Vite, deployed to Vercel.

**Live site:** https://docs-site-kappa-coral.vercel.app

## Development

```bash
bun install
bun run dev      # http://localhost:5173
bun run build    # production build to dist/
```
